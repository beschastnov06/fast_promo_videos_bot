# Telegram Video Bot MVP

MVP Telegram-бота, который принимает видео, приводит его к вертикальному формату, добавляет верхнюю рекламную плашку и нижние субтитры, затем возвращает готовый MP4.

## Возможности

- Принимает Telegram video до `20 МБ`.
- Приводит видео к `9:16`, `1080x1920`, без растягивания и без жесткой обрезки.
- Горизонтальные видео вписываются в вертикальный кадр с черными полями сверху и снизу.
- Вертикальные видео аккуратно подгоняются под `9:16`; если пропорция отличается, добавляются черные поля.
- Поддерживает верхнюю рекламную плашку как текст или картинку-баннер.
- Поддерживает PNG-баннеры с прозрачным фоном, если отправить картинку файлом без сжатия.
- После отправки видео бот отдельным сообщением просит рекламный текст или баннер.
- Рекламный контент применяется только к тому видео, после которого он был отправлен.
- Если рекламного контента нет, пользователь может нажать кнопку `Без контента`.
- Текст сверху ограничен `60` символами и максимум `2` строками.
- Речь из видео автоматически распознается и добавляется снизу как субтитры.
- Возвращает результат как Telegram video.
- На этапе разработки доступ ограничивается whitelist по Telegram username.

## Видео-профиль MVP

- Контейнер: `MP4`.
- Размер: `1080x1920`.
- Соотношение сторон: `9:16`.
- Частота кадров: `30 fps`.
- Видео: `H.264`, `yuv420p`, profile `main`, level `4.0`.
- Качество: `CRF 26`.
- Аудио: `AAC 128k`, `44100 Hz`.
- Безопасные отступы: снизу `330 px`, справа `160 px`, слева `160 px`.
- Входной лимит: `20 МБ`.

## Как добавить рекламную плашку

Макет итогового ролика:

```text
┌──────────────────────────────┐
│                              │
│  Рекламный текст или баннер  │
│                              │
├──────────────────────────────┤
│                              │
│                              │
│                              │
│          ВИДЕО 9:16          │
│                              │
│                              │
│                              │
├──────────────────────────────┤
│       Субтитры снизу         │
│   с безопасными отступами    │
└──────────────────────────────┘
```

Текстовая рекламная плашка сверху:

```text
Реклама: @example
```

Порядок:

1. Отправьте видео.
2. Бот попросит рекламный текст, баннер или предложит кнопку `Без контента`.
3. Отправьте текст.
4. Бот поставит этот текст в верхнюю плашку.

Картинка-баннер сверху:

1. Отправьте видео.
2. Бот попросит рекламный текст, баннер или предложит кнопку `Без контента`.
3. Отправьте картинку обычным фото или файлом без сжатия.
4. Бот поставит картинку в верхнюю плашку.

Если нужен PNG без фона, отправляйте баннер как файл без сжатия. Так Telegram сохранит прозрачность, а бот наложит баннер без белой подложки.

Если рекламного контента нет:

```text
Без контента
```

Рекламный контент не сохраняется для будущих видео. Для каждого нового ролика бот заново попросит текст, баннер или `Без контента`.

После рекламного контента бот может показать подтверждение параметров монтажа. На этапе MVP все параметры остаются значениями по умолчанию, но этот шаг готовит интерфейс к будущему редактированию.

Рекомендуемый текст:

```text
Настроим монтаж ролика?

Сейчас применю параметры по умолчанию:

Формат: 9:16, без растягивания
Заполнение пустоты: черное
Шрифт субтитров: DejaVu Sans
Зеркальность видео: нет

Если все подходит, нажмите "Смонтировать видео"
```

Рекомендуемые кнопки:

```text
Смонтировать видео
Изменить настройки
```

Для текущего MVP достаточно кнопки `Смонтировать видео`. Кнопку `Изменить настройки` можно добавить позже, когда появятся выбор шрифта, зеркальность, цветокор и другие параметры.

Чтобы отменить текущую обработку:

```text
/clear_ad
```

Нижние субтитры создаются автоматически из речи в видео. Они размещаются выше зоны интерфейса Reels/TikTok, чтобы не загораживать кнопки, описание и навигацию.

## Сообщения пользователю

Старт:

```text
Отправьте видео до 20 МБ, а я сделаю вертикальный ролик 1080x1920. После отправки видео я попрошу рекламный текст или баннер для верхней части макета, а субтитры снизу сделаю автоматически из аудио в видео.
```

Если пользователя нет в whitelist:

```text
на этапе разработки
```

Если команда `/ad` отправлена без текста:

```text
Напиши текст после команды, например: /ad Реклама: @example
```

Если команда `/ad` или баннер отправлены без активного видео:

```text
Сначала отправь видео, а потом рекламный текст или баннер для него.
```

Если текущая обработка отменена:

```text
Текущая обработка отменена.
```

Если активного видео для отмены нет:

```text
Нет активного видео для отмены.
```

Если видео больше лимита:

```text
Ошибка: видео слишком большое. Максимальный размер сейчас — 20 МБ.
```

После успешного приема видео:

```text
Видео принято 👌

Вы можете добавить рекламный контент (текст или баннер) сверху макета - отправьте то что необходимо

❗️❗️❗️формат баннеров .png необходимо отправить в формате "без сжатия"

Если дополнительного контента нет отправьте сообщение "Без контента"
```

Если пользователь нажал кнопку или отправил `Без контента`:

```text
Видео будет без рекламного контента
```

После отправки рекламного текста или баннера:

```text
Рекламный контент обрабатывается
```

Если рекламный контент подходит под условия:

```text
Рекламный контент обработан
```

После рекламного контента, перед монтажом:

```text
Настроим монтаж ролика?

Сейчас применю параметры по умолчанию:

Формат: 9:16, без растягивания
Заполнение пустоты: черное
Шрифт субтитров: DejaVu Sans
Зеркальность видео: нет

Если все подходит, нажмите "Смонтировать видео"
```

Перед монтажом:

```text
Начал монтировать видео
```

После успешной обработки:

```text
Готово
```

Если баннер не удалось скачать:

```text
Ошибка: не удалось скачать баннер. Попробуй другую картинку.
```

Если баннер больше лимита:

```text
Ошибка: баннер слишком большой. Максимум — 5 МБ.
```

Если рекламный текст слишком длинный:

```text
Ошибка: рекламный текст слишком длинный. Максимум — 60 символов.
```

Если произошла ошибка:

```text
Ошибка: <описание ошибки>
```

Если бот ждет рекламную плашку, но пользователь отправил неподходящее сообщение:

```text
Отправь рекламный текст сообщением, картинку-баннер или нажми «Без контента».
```

Если бот не ждет рекламную плашку и пользователь отправил не видео:

```text
Пожалуйста, отправь видео файлом Telegram video.
```

## Ограничения MVP

- Нет очередей задач.
- Нет базы данных.
- Нет админки.
- Нет платежей.
- Нет хранения истории обработок.
- Обработка выполняется прямо в процессе бота.
