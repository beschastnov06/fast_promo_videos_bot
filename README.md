# Telegram Video Bot MVP

MVP Telegram-бота, который принимает видео, приводит его к вертикальному формату, добавляет верхнюю рекламную плашку и нижние субтитры, затем возвращает готовый MP4.

## Возможности

- Принимает Telegram video до `20 МБ`.
- Приводит видео к `9:16`, `1080x1920`, без растягивания и без жесткой обрезки.
- Горизонтальные видео вписываются в вертикальный кадр с черными полями сверху и снизу.
- Вертикальные видео аккуратно подгоняются под `9:16`; если пропорция отличается, добавляются черные поля.
- Поддерживает верхнюю рекламную плашку как текст или картинку-баннер.
- Поддерживает PNG-баннеры с прозрачным фоном, если отправить картинку файлом без сжатия.
- После отправки видео бот отдельным сообщением просит рекламный текст или баннер.
- Рекламный контент применяется только к тому видео, после которого он был отправлен.
- Если рекламного контента нет, пользователь может нажать кнопку `Без контента`.
- После рекламного контента бот показывает редактируемый экран параметров монтажа.
- Текст сверху ограничен `60` символами и максимум `2` строками.
- Речь из видео автоматически распознается и добавляется снизу как субтитры.
- Возвращает результат как Telegram video.
- На этапе разработки доступ ограничивается whitelist по Telegram username.

## Видео-профиль MVP

- Контейнер: `MP4`.
- Размер: `1080x1920`.
- Соотношение сторон: `9:16`.
- Частота кадров: `30 fps`.
- Видео: `H.264`, `yuv420p`, profile `main`, level `4.0`.
- Качество: `CRF 26`.
- Аудио: `AAC 128k`, `44100 Hz`.
- Безопасные отступы: снизу `330 px`, справа `160 px`, слева `160 px`.
- Входной лимит: `20 МБ`.

## Как добавить рекламную плашку

Макет итогового ролика:

```text
┌──────────────────────────────┐
│                              │
│  Рекламный текст или баннер  │
│                              │
├──────────────────────────────┤
│                              │
│                              │
│                              │
│          ВИДЕО 9:16          │
│                              │
│                              │
│                              │
├──────────────────────────────┤
│       Субтитры снизу         │
│   с безопасными отступами    │
└──────────────────────────────┘
```

Текстовая рекламная плашка сверху:

```text
Реклама: @example
```

Порядок:

1. Отправьте видео.
2. Бот попросит рекламный текст, баннер или предложит кнопку `Без контента`.
3. Отправьте текст.
4. Бот поставит этот текст в верхнюю плашку.

Картинка-баннер сверху:

1. Отправьте видео.
2. Бот попросит рекламный текст, баннер или предложит кнопку `Без контента`.
3. Отправьте картинку обычным фото или файлом без сжатия.
4. Бот поставит картинку в верхнюю плашку.

Если нужен PNG без фона, отправляйте баннер как файл без сжатия. Так Telegram сохранит прозрачность, а бот наложит баннер без белой подложки.

Если рекламного контента нет:

```text
Без контента
```

Рекламный контент не сохраняется для будущих видео. Для каждого нового ролика бот заново попросит текст, баннер или `Без контента`.

Чтобы отменить текущую обработку:

```text
/clear_ad
```

Нижние субтитры создаются автоматически из речи в видео. Они размещаются выше зоны интерфейса Reels/TikTok, чтобы не загораживать кнопки, описание и навигацию.

## Ограничения MVP

- Нет очередей задач.
- Нет базы данных.
- Нет админки.
- Нет платежей.
- Нет хранения истории обработок.
- Обработка выполняется прямо в процессе бота.

## Параметры монтажа

После того как пользователь отправил рекламный текст, баннер или выбрал `Без контента`, бот отправляет одно сообщение с параметрами монтажа. Это сообщение редактируется при изменении настроек, чтобы не засорять чат.

```text
Параметры для монтажа:

Формат: 9:16, без растягивания
Заполнение пустоты: черное
Шрифт субтитров: DejaVu Sans
Цвет субтитров: белый
Зеркальность видео: нет

Если все подходит, нажмите "Отправить в монтаж"
```

Кнопки под сообщением:

```text
Изменить формат
Изменить цвет заполнения
Изменить шрифт субтитров
Изменить цвет субтитров
Включить зеркальность
Отправить в монтаж
```

Кнопки сделаны как inline-кнопки. При выборе параметра бот редактирует уже отправленное сообщение с параметрами. Отдельные сообщения для изменения параметров не создаются.

В Telegram нельзя надежно задать отдельный цвет кнопки. Поэтому кнопка отправки в монтаж выделяется текстом: `✅ Отправить в монтаж`.

### Изменение формата

На старте лучше дать пользователю не слишком много вариантов:

```text
9:16 — Reels / TikTok / Shorts
1:1 — квадратный пост
4:5 — лента Instagram
16:9 — горизонтальное видео
```

По умолчанию: `9:16`.

Для рекламных нарезок основной формат остается `9:16`, потому что он подходит под Reels, TikTok и Shorts. Остальные форматы нужны, если пользователь хочет переиспользовать ролик в ленте или на YouTube.

### Изменение цвета заполнения

Заполнение пустоты — это цвет полей, которые появляются, если видео не совпадает с выбранным форматом.

Доступные варианты:

```text
Черное
Белое
```

По умолчанию: `черное`.

Черное лучше подходит для большинства вертикальных роликов и выглядит привычнее в Reels/TikTok/Shorts. Белое можно оставить как вариант для более светлых рекламных макетов.

### Изменение шрифта субтитров

Шрифт по умолчанию: `DejaVu Sans`.

Доступные варианты:

```text
DejaVu Sans — базовый
Inter — чистый интерфейсный
Roboto — нейтральный
DejaVu Sans Mono — моноширинный
```

Для рекламных субтитров чаще всего используют жирные гротески: Inter, Roboto, Montserrat, Arial/Helvetica-подобные шрифты. Они хорошо читаются на телефоне, особенно с белым текстом и темной обводкой или подложкой.

`DejaVu Sans` уже есть в Docker-образе и стабильно работает с кириллицей. Для прод-выбора в Docker дополнительно устанавливаются `Inter` и `Roboto`. Когда добавим кастомные шрифты в проект, можно сделать `Montserrat` основным вариантом для более современного вида.

### Изменение цвета субтитров

Доступные варианты:

```text
Белый
Черный
```

По умолчанию: `белый`.

Если выбран белый цвет, бот использует темную подложку под субтитрами. Если выбран черный цвет, бот использует светлую подложку, чтобы текст оставался читаемым.

### Зеркальность видео

Зеркальность — это горизонтальное отражение видео.

Варианты:

```text
Нет
Да
```

По умолчанию: `нет`.

Кнопка `Включить зеркальность` может работать как переключатель: нажал один раз — стало `да`, нажал еще раз — снова `нет`. После переключения бот обновляет сообщение с текущими параметрами.
